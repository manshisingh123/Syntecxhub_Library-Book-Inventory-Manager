# Syntecxhub_Library-Book-Inventory-Manager
A simple Python-based Library Management System that allows users to manage books efficiently. The application provides features to add books, search by title or author, issue books, return books, and maintain library records using Python data structures and file handling.

This project demonstrates the use of Object-Oriented Programming (OOP), Python collections, and JSON file storage to simulate a real-world library management system.
